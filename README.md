# pdxscript.nvim

I had Claude create a Neovim plugin leveraging
[vic3-tiger](https://github.com/amtep/tiger) to provide diagnostics and looking
at many other projects with tree sitter grammar/etc to create syntax
highlighting for paradox script files in neovim.

I was just tired of editing plain .txt files when patching some mods for fun,
and wanted better editor support.

I probably won't maintain this.
